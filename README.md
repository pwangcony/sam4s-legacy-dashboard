# SAM4S 레거시 전환 대시보드

클래식 ASP(VBScript) 레거시 환경에서 Next.js 기반 모던 웹 스택으로의 아키텍처 전환을 한눈에 보여주는 비교 대시보드 시안입니다.

React 19 + TypeScript + Vite + Tailwind CSS 로 구성되어 있습니다.

## 로컬 실행

**사전 준비물:** Node.js

1. 의존성 설치:
   `npm install`
2. 개발 서버 실행:
   `npm run dev`
3. 브라우저에서 `http://localhost:3000` 접속

## 빌드

```
npm run build      # 프로덕션 빌드 (dist/)
npm run preview    # 빌드 결과 미리보기
npm run lint       # 타입 체크 (tsc --noEmit)
```
